## gwf-cli 0.1.4

Recommended install:

```bash
npm install -g gwf-cli@0.1.4 --registry https://registry.npmjs.org/
```

### Highlights

- Init enables Grok auto-load + commit scope check (confirm, default Yes)
- `gwf trust` / `gwf enable-automations`
- GitHub README: **English** | **中文** language switch
- User guide: `docs/使用说明.md` (Chinese)

### Quick start

```bash
cd your-project
gwf init -u your-name
grok
```
