## gwf-cli 0.1.3

**English** | **中文**

Grok Workflow Framework — use with Grok Build in any project.  
Grok 工作流框架 — 在任意项目中配合 Grok Build 使用。

### Install / 安装

```bash
npm install -g gwf-cli@0.1.3 --registry https://registry.npmjs.org/
```

### Quick start / 快速开始

```bash
cd your-project
gwf init -u your-name    # Enter = enable recommended setup / 直接回车 = 开启推荐配置
grok                     # Describe the goal / 直接说需求
```
