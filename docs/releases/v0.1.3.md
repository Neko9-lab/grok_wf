## gwf-cli 0.1.3

Grok Workflow Framework — 在任意项目中配合 Grok Build 使用。

### 安装

```bash
npm install -g gwf-cli@0.1.3 --registry https://registry.npmjs.org/
```

### 本版本亮点（0.1.0 → 0.1.3）

| 版本 | 内容 |
|------|------|
| **0.1.3** | 部署文档 `docs/部署与使用.md`、CHANGELOG |
| **0.1.2** | init 二次确认改为中文白话 |
| **0.1.1** | init 集成：Grok 自动加载 + git 提交范围检查（可确认） |
| **0.1.0** | 首发 npm：init/update/doctor/scope/Grok 适配 |

### 快速开始

```bash
cd your-project
gwf init -u 你的名字    # 直接回车 = 开启推荐配置
grok                   # 直接说需求
```
