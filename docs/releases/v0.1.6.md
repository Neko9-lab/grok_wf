### gwf-cli 0.1.6（推荐）

```bash
npm install -g gwf-cli@0.1.6 --registry https://registry.npmjs.org/
```

**本版要点**

- `gwf init`：可开启 Grok 自动加载 + 提交范围检查（确认后默认开启）
- GitHub 首页：**中文为主**，[English](https://github.com/Neko9-lab/grok_wf/blob/main/README.en.md)
- 使用说明：`docs/使用说明.md`

**快速开始**

```bash
cd 你的项目
gwf init -u 你的名字
grok
```

- 仓库：https://github.com/Neko9-lab/grok_wf  
- English intro: https://github.com/Neko9-lab/grok_wf/blob/main/README.en.md  
- npm：https://www.npmjs.com/package/gwf-cli
