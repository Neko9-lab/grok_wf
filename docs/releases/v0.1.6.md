### gwf-cli 0.1.6 (recommended)

```bash
npm install -g gwf-cli@0.1.6 --registry https://registry.npmjs.org/
```

- GitHub README language switch: **English** | **中文**

```bash

```

## 中文

### gwf-cli 0.1.6（推荐安装）

```bash
npm install -g gwf-cli@0.1.6 --registry https://registry.npmjs.org/
```

**本版要点**

- `gwf init` 可开启：Grok 自动加载工作流 + 提交前范围检查（二次确认，默认是）
- GitHub 首页 README：**English** | **中文** 切换
- 使用说明：`docs/使用说明.md`

**快速开始**

```bash
cd 你的项目
gwf init -u 你的名字
grok
```

- 仓库：https://github.com/Neko9-lab/grok_wf  
- npm：https://www.npmjs.com/package/gwf-cli  

> 说明：曾发布的 0.1.4 / 0.1.5 为文档噪音已从 npm 撤回；**0.1.6** 是 0.1.3 之后的干净版本。
